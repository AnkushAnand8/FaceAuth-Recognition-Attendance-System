# FaceAuth-Recognition-Attendance-System
FaceAuth Recognition Attendance System uses machine learning  algorithms to detect, recognize, and verify individuals’ faces in real-time  and automatically record attendance.This project demonstrates how computer vision techniques can be used to build a smart,efficient and automated attendance system using a camera and machine learning model
